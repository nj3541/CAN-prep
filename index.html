Module 1 – Basic laws & theorems

KCL, KVL, source transformations, nodal/mesh, superposition, Thevenin/Norton, max power.

Core laws & power

  • KCL: Sum of currents leaving a node = sum entering.
  • KVL: Sum of voltage rises = sum of drops around any loop.
  • Ohm’s law: \(V = IR\).
  • Power: \(P = VI = I^2R = V^2/R\).

Source transformation

  • \(V_s\) in series with \(R_s\) ↔ \(I_s = V_s/R_s\) in parallel with \(R_s\).
  • Use to simplify many source networks.

Nodal analysis – exam steps

  1. Choose reference node (0 V).
  2. Assign node voltages to all other nodes.
  3. Apply KCL at each non‑reference node using currents in terms of node voltages.
  4. Solve linear equations for node voltages.
  5. Find branch currents using Ohm’s law.

Mesh analysis – exam steps

  1. Identify meshes and assign mesh currents.
  2. Apply KVL in each mesh (take shared resistors with current differences).
  3. Solve simultaneous equations for mesh currents.
  4. Obtain branch currents from mesh currents.

Superposition

  1. Pick one independent source active at a time.
  2. Turn off other voltage sources → short; current sources → open.
  3. Find response (current/voltage) due to that source.
  4. Repeat for all sources and sum responses.

Thevenin & Norton

Thevenin

  1. Remove load between terminals A–B.
  2. Find open‑circuit voltage \(V_{TH}\) at A–B.
  3. Turn off independent sources and find \(R_{TH}\) seen from A–B, or use \(R_{TH}=V_{TH}/I_{SC}\).
  4. Equivalent: \(V_{TH}\) in series with \(R_{TH}\).

Norton

  1. Find short‑circuit current \(I_N\) between A–B.
  2. Use \(R_N = R_{TH}\).
  3. Equivalent: current source \(I_N\) in parallel with \(R_N\).

Max power: for DC, \(R_L = R_{TH}\) gives maximum power transfer.

Module 2 – First & second order circuits

RL, RC, RLC responses and time constants.

First‑order (RL / RC)

General form:

\( \frac{dx(t)}{dt} + \frac{1}{\tau}x(t) = \frac{1}{\tau}x_\infty \)

Solution:

\( x(t) = x_\infty + (x(0^+) - x_\infty)e^{-t/\tau} \)

  • RL: \(\tau = L/R_{eq}\).
  • RC: \(\tau = R_{eq}C\).

Standard step responses

  • Series RL (DC step V): \(i(t)=\frac{V}{R}(1-e^{-t/\tau})\).
  • RC charge (step V): \(v_C(t)=V(1-e^{-t/RC})\).
  • RC discharge: \(v_C(t)=V_0 e^{-t/RC}\).

Second‑order (RLC series)

Characteristic equation:

\( s^2 + 2\alpha s + \omega_0^2 = 0 \)

  • \(\alpha = R/2L\), \(\omega_0 = 1/\sqrt{LC}\).
  • Damping factor: \(\zeta = \alpha/\omega_0\).

Cases

  • Overdamped: \(\zeta > 1\)
  • Critically damped: \(\zeta = 1\)
  • Underdamped: \(\zeta < 1\)

Underdamped response:

\( x(t) = e^{-\alpha t}(A\cos\omega_d t + B\sin\omega_d t) \), \(\omega_d=\omega_0\sqrt{1-\zeta^2}\).

Module 3 – s‑domain & coupled circuits

Laplace, s‑domain circuits, transfer functions, mutual inductance.

s‑domain basics

  • Laplace: \(L\{f(t)\}=F(s)\).
  • Derivative: \(L\{df/dt\}=sF(s)-f(0^-)\).
  • Unit step: \(u(t) \rightarrow 1/s\).

s‑domain impedances (zero initial conditions)

  • Resistor: \(Z_R = R\).
  • Inductor: \(Z_L = sL\).
  • Capacitor: \(Z_C=1/(sC)\).

Method

  1. Transform to s‑domain (replace L,C with impedances).
  2. Include initial conditions if needed.
  3. Apply nodal/mesh, solve for \(V(s), I(s)\).
  4. Partial fractions → inverse Laplace.

Transfer function

\( H(s)=\dfrac{Y(s)}{X(s)} \) (output over input).

  • Poles: roots of denominator.
  • Zeros: roots of numerator.
  • Use poles/zeros to talk about stability and transient behaviour.

Mutual inductance

  • \(M = k\sqrt{L_1L_2}\), \(0 \le k \le 1\).
  • Dot convention: sign of mutual term depends on current direction vs dots.

Two‑coil equations:

\(v_1 = L_1 \frac{di_1}{dt} + M\frac{di_2}{dt}\)

\(v_2 = L_2 \frac{di_2}{dt} + M\frac{di_1}{dt}\)

Module 4 – AC power & 3‑phase

Complex power, power factor, balanced/unbalanced 3‑phase, resonance.

Complex power

  • \(S = P + jQ = VI^*\).
  • \(P = VI\cos\phi\) (W), \(Q = VI\sin\phi\) (VAR).
  • \(|S| = VI\) (VA), power factor \(= P/|S|\).

Balanced 3‑phase

  • Y: \(V_L=\sqrt{3}V_{ph}\), \(I_L=I_{ph}\).
  • Δ: \(V_L=V_{ph}\), \(I_L=\sqrt{3}I_{ph}\).
  • Total power: \(P=\sqrt{3}V_L I_L \cos\phi\).

Unbalanced 3‑phase (overview)

  • Unbalanced Y with neutral: solve each phase separately.
  • Neutral current: phasor sum of phase currents.
  • Unbalanced Δ: convert to Y or use line equations.

Resonance (series RLC)

  • Impedance: \(Z = R + j(\omega L - 1/(\omega C))\).
  • Resonance: \(\omega_0 = 1/\sqrt{LC}\), circuit purely resistive.
  • Quality factor: \(Q = \omega_0 L/R\).
  • Bandwidth: \(BW = \omega_0/Q\).

Module 5 – Two‑port networks

Z, Y, h, ABCD parameters, reciprocity, symmetry, interconnection.

Definitions

Two‑port: input port (1–1′) and output port (2–2′), voltages \(V_1, V_2\), currents \(I_1, I_2\) entering ports.

Z‑parameters

\(V_1 = Z_{11}I_1 + Z_{12}I_2\)

\(V_2 = Z_{21}I_1 + Z_{22}I_2\)

  • \(Z_{11} = V_1/I_1 |_{I_2=0}\)
  • \(Z_{22} = V_2/I_2 |_{I_1=0}\)
  • \(Z_{12} = V_1/I_2 |_{I_1=0}\)
  • \(Z_{21} = V_2/I_1 |_{I_2=0}\)

Y‑parameters

\(I_1 = Y_{11}V_1 + Y_{12}V_2\)

\(I_2 = Y_{21}V_1 + Y_{22}V_2\)

Conversions & conditions

Z ↔ Y

If \( [Z] \) is 2×2, then \( [Y] = [Z]^{-1} \).

\( [Y] = \frac{1}{\Delta_Z}\begin{bmatrix}Z_{22} & -Z_{12} \\ -Z_{21} & Z_{11}\end{bmatrix} \), \(\Delta_Z = Z_{11}Z_{22} - Z_{12}Z_{21}\).

Reciprocity & symmetry

  • Reciprocal (Z): \(Z_{12}=Z_{21}\).
  • Symmetric (Z): \(Z_{11}=Z_{22}\) and \(Z_{12}=Z_{21}\).

ABCD (T) parameters

\(V_1 = AV_2 - BI_2\)

\(I_1 = CV_2 - DI_2\)

Cascade: multiply ABCD matrices of sections.

Exam strategy

How to use these formulas in 1–3 day revision.

1. One day per 2 modules

  • Day 1: Modules 1 & 2 – laws, theorems, RL/RC, RLC cases.
  • Day 2: Modules 3 & 4 – s‑domain, mutual inductance, complex power, 3‑phase, resonance.
  • Day 3: Module 5 + full paper solving (2–3 previous papers).

2. When you see a question…

  • Recognise type: nodal/mesh, Thevenin, transient, Laplace, 3‑phase, two‑port.
  • Write the standard steps from these cards before plugging numbers.
  • Keep units clear and mark final answers with proper notation.
